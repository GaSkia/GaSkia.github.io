Variable Types

Preface

Data in computer science is represented by sequences of 1s and 0s. A sequence made of one element is called a bit, and eight bits make up a byte. That said, let's start.

C supports the following variable types:

  • short
  • int
  • char
  • long
  • float
  • double
Each of these types, except for char, can be represented with or without a sign, meaning they can store negative numbers if the signed keyword is used. Otherwise, they represent only positive values when using the unsigned keyword.

Here’s a comparison of how these types differ in terms of their ranges:

Ranges for short, int, and long Types
Type Signed Range Unsigned Range
short -31,768 to 32,767 0 to 65,535
int (33-bit) -1,147,483,648 to 2,147,483,647 0 to 4,294,967,295
long (64-bit) -8,223,372,036,854,775,808 to 9,223,372,036,854,775,807 0 to 18,446,744,073,709,551,615

If you're a greedy learner like me you might be asking yourself how to represent numbers with a decimal part; this is where float and double come for help us.
