The Fundamentals Of Low-Level Programming in C

Even though C is not very popular in these days it's still a good place to start, it teaches you how a computer works, how to manage memory and raw performance.

Variables Type.

Preface.

Data in computer science is represented by sequences of 0 and 1, a sequence made of one element is called a bit and eight bits made up a byte. That said we can start.

Standard C89 libraries supports these variable types:

  • short
  • int
  • char
  • long
  • float
  • double
Every of these type excpet char can be represented with or without sign, this means that if you don't specify it, they can also store negative numbers. short, int and long all represent integers but with a neat difference. short uses 16 bits (2 bytes), storing values from 0 to 65,535 when unsigned when you don't use the unsigned keyword.
